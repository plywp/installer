🚀 PlyWP Installer

Simple, fast installer for PlyWP Panel + Plyorde Daemon.

⚡ Install
```bash
bash <(curl -s http://plywp-installer.urixen.io/install.sh)
```
📦 Requirements
Debian-based system (Ubuntu, Debian)
Root access
🧠 What it does
Installs PlyWP panel
Installs Plyorde daemon
Sets up required dependencies
Configures system services
